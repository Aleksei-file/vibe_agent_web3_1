# vibe_agent_web3_1
-
Проект: DeFi Insight Dashboard
Концепция
Это дашборд, где пользователь может подключить кошелек, увидеть свои "псевдо-активы" в протоколе, попробовать "стейкать" токены и посмотреть историю транзакций. Поскольку бэкенда нет, мы используем статические моки, но взаимодействие с кошельком будет настоящим.
Основной функционал
Wallet Connection: Подключение через MetaMask/WalletConnect (Wagmi hooks).
Account Overview: Отображение адреса, баланса сети и статуса подключения.
Mock Yield Farming: Интерфейс для стейкинга (кнопки Approve/Deposit), который вместо реального смарт-контракта будет вызывать всплывающее окно подписи в кошельке.
Transaction Feed: Список "прошедших" транзакций из статического JSON.
